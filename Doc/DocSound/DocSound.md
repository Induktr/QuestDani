# Аудио-Гид "Wisdom Forge" (Звук как Инструмент Вдохновения)

### 1. Введение: Звук как Часть Ценности

Звуковая система "Wisdom Forge" — это не просто фон. Это **инструмент для создания атмосферы и усиления эмоционального отклика**, который помогает донести **ценность (Принцип 2)** полученной мудрости.

**Принципы:**
*   **Качество > Количество (Принцип 3):** Лучше один, но запоминающийся и чистый звук, чем десять посредственных.
*   **Здоровье (Принцип 5):** Звуки не должны быть раздражающими. Пользователь всегда может их отключить.

### 2. Категории Звуков

| Категория | Назначение | Описание |
|:---|:---|:---|
| **Интерактивные** | Аудио-фидбек на действия | Короткие, минималистичные звуки для подтверждения действий (клик, выбор). |
| **Атмосферные** | Создание фона для концентрации | Спокойная, медитативная, почти неслышимая музыка во время "Испытаний". |
| **Награды** | Усиление чувства достижения | Яркий, позитивный, кристальный звон при получении "Кристалла" или "Короны". |
| **Голос Мастера** | Вдохновение и мотивация | Короткие, мудрые фразы, произнесенные голосом, похожим на голос "Мастера". |

### 4. Техническая Реализация

*   **Хранение:** Все аудио-файлы хранятся в облачном хранилище.
*   **Воспроизведение:** Используется стандартный HTML5 Audio API на фронтенде.
*   **"Голос Мастера":** Для MVP могут быть использованы **высококачественные, сгенерированные ИИ голоса**, которые похожи на голоса реальных людей, чтобы избежать проблем с авторским правом. Это **прагматичное (Принцип 12)** решение.

### 5. Риски
*   **Авторское право:** Использование реальных голосов требует разрешения. На начальном этапе следует использовать **сгенерированные ИИ голоса** или нанять дикторов.
